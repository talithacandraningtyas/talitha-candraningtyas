Tentang Saya & Tujuan Website

"Setiap baris kode adalah langkah maju menuju pemahaman yang lebih baik tentang dunia digital."

Tujuan Website Ini

Website portofolio ini dibuat sebagai bagian dari tugas mata kuliah Web Development sekaligus sebagai arsip digital. Tujuan utamanya adalah:

  • Arsip Pembelajaran: Mengumpulkan semua tugas dan latihan HTML/CSS dari Pertemuan 1 hingga 21 di satu tempat yang mudah diakses.
  • Showcase Kemampuan: Menunjukkan perkembangan keterampilan *coding* dasar yang telah saya pelajari.
  • Dokumentasi: Sebagai alat presentasi visual kepada dosen atau calon rekan kerja mengenai kemampuan teknis dasar saya.

Keahlian yang Dipelajari

Selama proses pembuatan portofolio ini dan di mata kuliah terkait, saya fokus mempelajari beberapa *skill* dasar web:

  • HTML5 (Struktur Dasar)
  • CSS Dasar (Styling dan Layout)
  • CSS Box Model
  • JavaScript Dasar (Fungsi, Variabel)
  • Git & GitHub (Version Control)
  • Pembuatan Form
  • Konsep Layout (Table & Div/Float)

Motivasi dan Harapan

Motivasi utama saya adalah mengubah teori menjadi praktik. Saya berharap di masa depan dapat mengembangkan kemampuan ke tingkat yang lebih mahir, seperti menggunakan kerangka kerja (framework) CSS modern dan JavaScript *library* untuk membangun aplikasi web yang lebih kompleks dan responsif.
